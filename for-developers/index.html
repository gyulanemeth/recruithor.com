How it woks

We continuously search for companies using your favorite technologies.

In order to confidently offer you to our partner companies, we will need to test your problem-solving skills, your technical knowledge, and last but not least, your algorithmic thinking. We test these things in a highly unusual process, that starts with a brain-teaser, and you will love.

Currently, we only test for the following frontend technologies: Javascript & Vue.js. We will very soon extend this list with the coolest frameworks, and other programming languages.

If you are a senior developer, then this process is not for you, because our testing process is optimized to test beginner, junior and medior developers.

We only hire people who are willing to (or want to) work remotely. Let us know your preferred time-zones, so we can pair you up with the proper companies.
